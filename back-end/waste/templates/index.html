Welcome to the AMH Project 0.2 API

This API provides image classification functionalities and system health checks.

Available Endpoints

  • /health - Check if the service is running
  • /status - Check if the model is loaded successfully
  • /class_names - List available classes for prediction
  • /logs - Retrieve recent system logs

Usage

To predict an image class, send a POST request to /predict with JSON body containing image_path.

{
  "image_path": "path/to/your/image.jpg"
}

To upload and predict multiple images, use the /upload endpoint as a POST request with form-data containing files.
